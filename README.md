# bs-react-intl

BuckleScript bindings to [react-intl](https://github.com/yahoo/react-intl).

## Installation
```shell
# yarn
yarn add bs-react-intl

# or npm
npm install --save bs-react-intl
```

## Examples
Clone this repo.

```shell
yarn install
yarn run build
```

Then open some `*.html` from [`examples`](./examples) folder.
